CSS interno

Um CSS interno é usado para definir um estilo para uma única página HTML.

Um CSS interno é definido na <head> seção de uma página HTML, dentro de um <style>elemento.

O exemplo a seguir define a cor do texto de TODOS os <h1> elementos (naquela página) como azul e a cor do texto de TODOS os <p> elementos como vermelho. Além disso, a página será exibida com uma cor de fundo "powderblue":
