HTML Image

Sintaxe

Observação:

Sempre especifique a largura e a altura de uma imagem. Se a largura e a altura não forem especificadas, a página da Web poderá piscar enquanto a imagem é carregada.

<img src="url" alt="alternatetext">

Largura e Altura ou Estilo?

Os atributos width, height e style são todos válidos em HTML.

No entanto, sugerimos o uso do atributo style. Isso evita que as folhas de estilo alterem o tamanho das imagens:

<img src="url" alt="alternatetext" style="width:000px; height:000px;">

Imagem flutuante

Use a propriedade CSS floatpara deixar a imagem flutuar à direita ou à esquerda de um texto:

<img src="url" alt="alternatetext" style="float:left; width:000px; height:000px;">
