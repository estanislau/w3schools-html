Atributo: style
Propriedade: background-image, background-repeat, background-size

Para adicionar uma imagem de plano de fundo em um elemento HTML, use o styleatributo HTML e a propriedade CSS background-image.

Se a imagem de fundo for menor que o elemento, a imagem se repetirá, horizontal e verticalmente, até chegar ao final do elemento

Para evitar que a imagem de fundo se repita, defina a propriedade background-repeat como no-repeat.

Se você deseja que a imagem de plano de fundo cubra todo o elemento, pode definir a background-sizepropriedade como cover.
