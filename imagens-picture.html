Elemento: <picture><source>
Atributo: srcset, media

Nota: Sempre especifique um <img> elemento como o último elemento filho do <picture> elemento. O <img> elemento é usado por navegadores que não oferecem suporte ao <picture> elemento ou se nenhuma das <source>tags corresponder.

Redimensionar o navegador para ver diferentes versões do carregamento da imagem em diferentes tamanhos de viewport. O navegador procura o primeiro elemento de origem em que a consulta de mídia corresponda à largura atual da janela de visualização do usuário, e busca a imagem especificada no atributo srcset.

O elemento img é necessário como a última tag filho do bloco de declaração de imagem. O elemento img é usado para fornecer compatibilidade com versões anteriores para navegadores que não suportam o elemento picture ou se nenhuma das tags de origem corresponder.

Observação: o elemento de imagem não é suportado no IE12 e anterior ou no Safari 9.0 e anterior.
