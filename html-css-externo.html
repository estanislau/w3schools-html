CSS externo

Uma folha de estilo externa é usada para definir o estilo de muitas páginas HTML.

Para usar uma folha de estilo externa, adicione um link para ela na <head> seção de cada página HTML:

This is a heading

This is a paragraph.

A folha de estilo externa pode ser escrita em qualquer editor de texto. O arquivo não deve conter nenhum código HTML e deve ser salvo com a extensão .css.
