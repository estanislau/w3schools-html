Os links HTML podem ser usados ​​para criar marcadores, para que os leitores possam pular para partes específicas de uma página da web.

Criar um marcador em HTML

Os marcadores podem ser úteis se uma página da Web for muito longa.

Para criar um marcador - primeiro crie o marcador e, em seguida, adicione um link a ele.

Quando o link for clicado, a página rolará para baixo ou para cima até o local com o marcador.

Primeiro, use o atributo id para criar um marcador.

Em seguida, adicione um link ao marcador ("Jump to Chapter 4"), dentro da mesma página:
